# Python Code for Algorithm Solving and File Automation

This is a simple Python project I made to practice coding algorithms, working with file systems, and parsing text strings using Regular Expressions (re module). It also helps me practice using Git and GitHub for tracking my code files.

## What is in this project?

The project has two main Python files:

1. **`algorithms.py`**
   * **Binary Search:** A fast search algorithm to find an item in a sorted list by splitting it in half every time.
   * **Two Sum:** A function that looks through a list of numbers to find which two values add up to a specific target number using a Python dictionary.

2. **`automation_tools.py`**
   * **File Finder:** Walks through folders and finds files that match a specific file extension (like `.py`).
   * **Log Parser:** Takes strings of text from system log files and extracts the date, error type, and message using basic Regex patterns.
